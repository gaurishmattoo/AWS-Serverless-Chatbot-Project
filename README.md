# AWS Serverless Chatbot for Customer Support

## 📌 Project Overview
This project demonstrates a serverless customer support chatbot using AWS Amazon Lex.

The chatbot can:
- Understand customer queries
- Process intents
- Handle slot values
- Respond automatically

---

# 🚀 AWS Services Used

- Amazon Lex
- AWS Lambda
- AWS IAM
- AWS CloudWatch

---

# 🏗️ Project Workflow

1. Create chatbot using Amazon Lex
2. Create intents
3. Add sample utterances
4. Configure slots
5. Build chatbot
6. Test chatbot responses
